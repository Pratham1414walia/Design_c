# Design_c-
LLD in C

A Modal Class is a class whose objects can move through different states during their lifetime.

But the class itself does not change — only the state of the object changes.

Simple Definition

👉 Modal Class = A class whose objects can exist in different states.

The object changes state over time, depending on events or actions.

Simple Example 1: TV

Class: TV

States:

OFF

ON

Standby

In UML:

❌ Objects should not change class
✔ Objects should change state
